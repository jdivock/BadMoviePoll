### Installation

You need JSPM installed globally:

```sh
$ npm i -g jspm
```

```sh
$ jspm install
```

Then run it however you feels ([live-server] is pretty awesome)

```
$ live-server
``` 

or good ole python

```
$ python -m SimpleHTTPServer
```
